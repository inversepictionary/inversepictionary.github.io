Inverse Pictionary

Welcome to Version 2.1.0

Game Overwiew

Last updated on Jan 25, 2020

This is an online game like pictionary but inverted. Instead of drawing a picture which others will guess, you will draw pictures others have described. As of the current update you can perfrom 3 tasks in any order:

- Describe images that are present in our database (click the Describe button bellow to begin)
- Draw images other people have described (click the Draw button bellow to begin)
- Judge other peoples drawings and descriptions (click the Judge button bellow to begin)

I hope you have a blast!

Note: DO NOT draw a picture for your own description and DO NOT judge your own description or drawing (In these cases refreshing the page will genrate a new task)
