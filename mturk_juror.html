Welcome to InversePictionary!

IMPORTANT! This study can ONLY run on a desktop/laptop

In this study you will be playing an online game like pictionary but inverted.
Instead of drawing a picture which others will guess,
you will describe pictures others will draw and draw pictures others have described.
This is the Judge Task: Judge an image and description pair collected from other users
Note:
- DO NOT draw a picture of your own description
- DO NOT judge your own description or drawing
(In these cases refreshing the page will genrate a new task)
The study is expected to take 5 minutes
You will describe 5 images
Thanks for participating!

Instructions

Firstly, you will evaluate the image-description pair based on how accurately the drawing reflects the actual image. Judge then based on similarity of visual characteristics (i.e. physical features and environment). This will be on a 5 point scale, where 5 means most similar and 1 means not similar at all. Secondly, you will evaluate the validity of the description. A description is invaild if it uses any descriptive features not explicitly seen in the image or if the description contains any other animal names. Have fun and Good Luck!
