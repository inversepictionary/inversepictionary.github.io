Welcome to InversePictionary!

IMPORTANT! This study can ONLY run on a desktop/laptop

In this study you will be playing an online game like pictionary but inverted.
Instead of drawing a picture which others will guess,
you will describe pictures others will draw and draw pictures others have described.
This is the Describe Task: Describe an image sampled from our database
Note:
- DO NOT draw a picture of your own description
- DO NOT judge your own description or drawing
(In these cases refreshing the page will genrate a new task)
The study is expected to take 5 minutes
Thanks for participating!

Instructions

The object is simple, you will be given 15 textboxes to describe features of the generated animal image. You will be evaluated on how accurately people can draw the image based on your description. Refrain from using any descriptive features not explicitly seen in the image. Also, your description should not contain the animals name or any other animal names. Lastly, DO NOT use unnessary verbage (ex. "the animal has pointed-ears", "it has a bushy tail", "this picture contains brown fur", etc.) instead ONLY write the attribute (ex. pointed-ears, bushy tail, brown fur, etc.). Failure to follow the previously mentioned rules will lead to disqualification. Have fun and Good Luck!
